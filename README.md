# Halaman Web Ulang Tahun Dea Putri Andini 💖

Halaman web ucapan selamat ulang tahun interaktif bernuansa pink-putih yang didesain khusus secara mobile-first (Instagram Stories style) untuk **Dea Putri Andini** dari **Arif**.

## Fitur Utama
- **Instagram Story Navigation**: Navigasi slide progres otomatis dengan sistem ketuk (tap kiri/kanan) layaknya Instastory.
- **Web Audio API Synth**: Suara kotak musik romantis yang memainkan lagu *Happy Birthday* dan *Canon in D* langsung melalui pensintesis peramban tanpa memerlukan koneksi internet.
- **Tiup Lilin Interaktif**: Lilin ulang tahun virtual yang dapat ditiup (padam) saat diklik untuk menampilkan doa rahasia.
- **Polaroid Memory Gallery**: 3 foto kenangan dengan bingkai polaroid estetik.
- **Surat Cinta Typewriter**: Efek ketik otomatis surat romantis dari Arif saat dibaca.
- **Interaksi Cinta**: Widget Love Meter tak terbatas dan pengambilan kartu janji secara acak.

## Cara Mengaktifkan Halaman Web di GitHub Pages
1. Masuk ke tab **Settings** repositori ini di GitHub.
2. Pilih menu **Pages** di kolom menu sebelah kiri.
3. Pada bagian **Build and deployment**, ubah Branch dari **None** menjadi **main** lalu klik **Save**.
4. Link situs Anda akan aktif beberapa saat kemudian di: `https://valdistatilamaeo.github.io/hbd-andini/`
